## Description

This repository is a template for a Typescript project. This template is already configured with:
 - Nodemon
 - Jest
 - Typescript
 - Eslint

## Install

```sh
npm install
```

## Run Application

#### Run dev mode
```sh
npm run dev
```

#### Build for production
```sh
npm run build
```

## Tests

#### Run unit tests
```sh
npm run unit-test
```

#### Run integration tests
```sh
npm run integration-test
```

#### Run all tests
```sh
npm run test
```
## Build

#### Run production
```sh
npm run start
```
